# Bank Project

This beginner-friendly Java project simulates a simple banking system using Object-Oriented Programming (OOP) concepts.

## Features
- `BankAccount` class with attributes for account number, account holder, and balance.
- Constructor to initialize bank accounts.
- Methods for deposit, withdrawal, and checking balance.
- Demonstrates creating objects, calling methods, and updating object state.

## How to run
1. Compile `BankAccount.java` and `Main.java`.
2. Run `Main.java` to see example transactions and account balances.

## Example Output
- Creates multiple accounts.
- Performs deposits and withdrawals.
- Displays current account balances.
